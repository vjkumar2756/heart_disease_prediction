# Heart Disease Prediction ML Project

## Overview
This project predicts the risk of heart disease using multiple Machine Learning algorithms.

It includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training
- Model evaluation


---

Installation

```bash
git clone [https://github.com/your-username/heart-disease-prediction.git](https://github.com/vjkumar2756/heart_disease_prediction.git)
cd heart-disease-prediction
pip install -r requirements.txt


Machine Learning Models Used
•Logistic Regression
•Support Vector Machine (SVM)
•Decision Tree
•Naive Bayes
•K-Nearest Neighbors (KNN)

Workflow
•Load dataset
•Clean data (handle missing/zero values)
•Encode categorical features
•Scale features
•Train models
•Evaluate models
•Save best model

Results
•Accuracy, Precision, Recall, F1-score comparison of all models
•Best model is saved in models/

Output
•Plots saved in outputs/plots/
•Reports saved in outputs/reports/
